## 🚀 Quick Start

### Installation

1. **Download** the latest release from [GitHub Releases](https://github.com/789Abhi/CCC-Plugin/releases)
2. **Upload** to your WordPress `/wp-content/plugins/` directory
3. **Activate** the plugin through WordPress admin
4. **Navigate** to `Custom Components` in your admin menu

### Create Your First Component
Go to Custom Components → Add New
Enter component name: "Hero Section"
Add fields: Title, Description, Button Text
Save component
Assign to any page/post
Configure field values
View on frontend

## 💡 How It Works

### 1. **Create Components**
Design your component structure with custom fields through the intuitive admin interface.

### 2. **Generate Templates**
Plugin automatically creates PHP templates in your theme's `/ccc-templates/` directory.

### 3. **Assign to Pages**
Use the meta box in post/page editor to add and configure components.

### 4. **Multiple Instances**
Add the same component multiple times with different content on any page.

### 5. **Frontend Display**
Components render automatically with your custom field values.
